dataprep.eda.missing.computation module¶

This module implements the plot_missing(df) function’s calculating intermediate part

dataprep.eda.missing.computation.plot_missing(pd_data_frame: pandas.core.frame.DataFrame, x_name: Optional[str] = None, y_name: Optional[str] = None, return_intermediate: bool = False) → Union[bokeh.plotting.figure.Figure, bokeh.models.layouts.Tabs, Tuple[Union[bokeh.plotting.figure.Figure, bokeh.models.layouts.Tabs], Any]][source]¶
Parameters

pd_data_frame – the pandas data_frame for which plots are calculated for each

column. :param x_name: a valid column name of the data frame :param y_name: a valid column name of the data frame :param return_intermediate: whether show intermediate results to users :return: A dict to encapsulate the intermediate results.

This function is designed to deal with missing values

There are three functions: plot_missing(df), plot_missing(df, x) plot_missing(df, x, y)

match (x_name, y_name)

case (Some, Some) => histogram for numerical column, bars for categorical column, qq-plot, box-plot, jitter plot, CDF, PDF case (Some, None) => histogram for numerical column and bars for categorical column case (None, None) => heatmap otherwise => error
