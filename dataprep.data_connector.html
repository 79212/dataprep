class dataprep.data_connector.types.Orient[source]¶

Bases: enum.Enum

Different types of table orientations ref: (https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.to_json.html). Currently, DataConnector supports two different types of orientaions:

  1. Split, which is column store.

  2. Records, which is row store.

Details can be found in the pandas page.
