column. :param x_name: a valid column name of the data frame :param y_name: a valid column name of the data frame :param value_range: range of value :param k: choose top-k element :param return_intermediate: whether show intermediate results to users :return: A (column: [array/dict]) dict to encapsulate the intermediate results.

match (x_name, y_name, k)

case (None, None, None) => heatmap case (Some, None, Some) => Top K columns for (pearson, spearman, kendall) case (Some, Some, _) => Scatter with regression line with/without top k outliers otherwise => error
